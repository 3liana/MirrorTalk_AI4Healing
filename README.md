# 光影识人 - Mirror Me

## AI自我觉察对话工具

> 通过与电影角色的AI对话，帮助你看见习惯性的思维模式，找回被忽略的真实自我。

一个基于Vue3 + LangChain.js的沉浸式AI对话应用，帮助用户通过与虚构电影角色的对话来进行自我觉察。

### 💡 我们解决什么问题

很多人在生活中感到"做自己好累"，却说不清楚问题出在哪里：

- 🤔 总觉得自己"应该"更努力/更温柔/更独立，但不知道这些"应该"从哪来
- 😔 习惯性压抑真实感受，担心"不合适"、"太敏感"、"会麻烦别人"
- 🔄 重复同样的模式（关系里、工作中、自我要求上），但意识不到
- 🚪 想要改变，却不知道从哪里开始

**核心痛点**：
- **自我觉察的门槛太高**：心理咨询太正式，自我反思不知道问自己什么
- **缺乏安全的探索空间**：直接面对"真实的自己"太有压力
- **传统测评的局限**：MBTI/九型人格给你标签，但不帮你理解"为什么"

### 🎯 我们的解决方案

**核心洞察**：当你和"虚构角色"对话时，心理防御会降低——你不是在被评判，而是在"借角色的嘴"说出自己的困惑。

**如何做到**：

1. **借用电影IP的安全距离** - 不是"你有什么问题"，而是"电影里这个角色的困惑，你有共鸣吗？"
2. **用对话代替问卷** - 不是选ABCD，而是自然聊天中，AI追问你的真实感受
3. **从觉察到行动** - 不只告诉你"你是什么样的人"，更给你"下一步可以尝试什么"

### 🎭 产品定位

| 维度 | 我们是 | 我们不是 |
|------|--------|----------|
| **核心目标** | 自我觉察工具 | 心理治疗/人生导师 |
| **体验感受** | 被理解、被看见 | 被诊断、被教育 |
| **使用场景** | 感到困惑时的陪伴 | 每日打卡的工具 |
| **用户关系** | 一面诚实的镜子 | 权威的建议者 |
| **内容风格** | 温暖、好奇、追问 | 分析、说教、结论 |

### 👥 适合谁使用

**核心用户**：希望建立自我认知、进行自我觉察的人

**具体画像**：
- 📍 **年龄**：22-35岁
- 🧠 **特征**：有一定反思能力，对心理学/哲学有兴趣
- 💼 **场景**：生活/工作中遇到重复的困扰，想要理解自己
- 🎬 **习惯**：看电影/剧时会共鸣，喜欢有深度的内容

**典型使用场景**：
- "最近总觉得很累，但说不清为什么"
- "我知道自己该拒绝，但就是说不出口"
- "又和伴侣/家人吵了同样的问题，我是不是有问题？"
- "MBTI测完了，然后呢？"

### 核心特性

- **沉浸式对话旅程**：15-20分钟的互动式对话体验，而非传统聊天窗口
- **打字机效果**：角色文本逐字出现，营造电影般的沉浸感
- **3个模块渐进探索**：
  - 共鸣进入（建立安全距离）
  - 温和追问（识别"应该"和真实需求）
  - 行动落地（转化为可执行的micro-action）
- **智能报告生成**：生成个性化的Mirror Report，包含思维模式分析和3个可执行的行动建议


### 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **状态管理**：Pinia
- **数据验证**：Zod
- **AI编排**：LangChain.js

### 快速开始

#### 1. 安装依赖

```bash
npm install
```

#### 2. 配置环境变量

```bash
# 复制 .env.example 到 .env.local
cp .env.example .env.local

# 编辑 .env.local，添加你的 OpenAI API Key
VITE_OPENAI_API_KEY=sk-xxx
```

#### 3. 启动开发服务器

```bash
npm run dev
```

打开浏览器访问 `http://localhost:5173`

### 项目结构

```
src/
├── roles/              # 预定义角色
├── journey/            # 类型定义和状态
├── chain/              # LangChain 引擎
├── llm/                # LLM 客户端
├── stores/             # Pinia 状态管理
├── components/         # Vue 组件
├── views/              # 页面视图
├── router.ts           # 路由
├── main.ts             # 入口
└── styles.css          # 全局样式
```

### 核心流程

1. 用户在首页选择角色
2. 初始化沉浸式对话旅程（12轮）
3. 每轮展示角色文本 + 用户交互（选择/评论）
4. 旅程结束后生成个性化 Mirror Report
5. 报告包含思维模式分析、3个micro-action、温暖的收尾信

### 环境变量

```
VITE_OPENAI_API_KEY=sk-your-key      # 必须
VITE_OPENAI_API_BASE=https://...    # 可选，默认 OpenAI
VITE_OPENAI_MODEL=gpt-3.5-turbo     # 可选
```
